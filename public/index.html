SSA Simple Node Server


Available API endpoints

  • /api/v1/generateUsers?userCount=5&maxPosts=10
    userCount = Number of users to generate (>0);
    maxPosts = API will generate random posts, from 0 to maxPosts number (>0);
  • /api/v1/users?page=1&per=3
    page = Page number (>0);
    per = Results per page (>0);
